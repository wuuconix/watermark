# watermark
大三上课设 数字水印

+ fomantic UI 写可视化界面
+ flask 做后端来调用blind_watermark库

---

快速开启服务：

```bash
#安装依赖
pip3 install -r ./requirements.txt
#运行python程序
python3 app.py
#另起一个终端，进入nginx目录
cd nginx
#开启nginx
start ./nginx
```

浏览器中打开 localhost 即可
